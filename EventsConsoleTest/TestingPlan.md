# План тестирования

## Этапы тестирования
### 1. Модульное тестирование
- **EventTests**:
  - Validate: проверка валидации полей.
  - ToString: проверка формата вывода.
- **TextExportServiceTests**:
  - ExportEvents: создание файла, обработка пустого списка.

### 2. Интеграционное тестирование
- Проверка взаимодействия EventService с EventsRepository и TextExportService.
- Сценарии: добавление, обновление, экспорт, уведомления.

### 3. Пользовательское тестирование
- Сценарии:
  - Добавление встречи (корректные/некорректные данные).
  - Редактирование, удаление, фильтрация, экспорт, уведомления.
- Проверка читаемости, обработки ошибок.

### 4. Нагрузочное тестирование
- Проверка производительности на 100 встречах.
